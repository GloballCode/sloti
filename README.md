## Sloti
A scheduling software for optimized organization

## 📚 Theme
This is a scheduling software, built to help you or your business to always know what and when to do! 
Works specially well for small businesess, like hair salons, with multiple workers.

## ✨ Functionalities
- Scheduling agenda from 8AM to 18PM
- Multiuser platform: connect all employees in one agenda, and keep track of all tasks for today, with no risks of someone scheduling something at the same time
- Quick and Simple: No unnecesary features. No ads. Just you, your team and your schedule.

## 🛠 Technologies
- HTML 
- CSS 
- JavaScript 
- Firebase Auth - for authentication 
- Firestore - for database and backend
- Firebase Hosting - for website hosting

## 🎯 Objective
This software was developed for a very specific reason, y'know. A hair salon sometimes had problems, because the eyebrows designer sometimes scheduled appointments the same day and time as the hairdresser, and so did the hairdresser. That's when the idea for Sloti showed up: a real-time multiuser scheduler, so no one will schedule things the same hour and date as the other employee.
